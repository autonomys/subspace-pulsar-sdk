Expand description

An error type for SS58 decoding.

Variants

BadBase58

Base 58 requirement is violated

BadLength

Length is bad

InvalidPrefix

Invalid SS58 prefix byte

FormatNotAllowed

Disallowed SS58 Address Format for this datatype

InvalidChecksum

Invalid checksum
